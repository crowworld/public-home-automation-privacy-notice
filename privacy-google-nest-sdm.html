Privacy Policy – Google Nest Device Access Integration

Last updated: 23 January 2026

Overview

This privacy policy describes how data is accessed and used by a private home automation system that integrates with Google Nest devices using the Google Smart Device Management (SDM) API.

This integration is non-commercial and is intended for use only within a single private household.

Data accessed

With explicit user consent, this system may access data exposed by Google Nest devices via the SDM API, which may include:

  • Device identifiers and metadata
  • Sensor readings (for example temperature, humidity, or device state)
  • Device status and operational information supported by the SDM API

No data outside what is provided by the SDM API is accessed.

Purpose of data access

The data is used solely for:

  • Private home automation
  • Monitoring and control of household devices
  • Informational display and internal system logic

The data is not used for advertising, profiling, third-party analytics, or resale.

Data storage and retention

  • Data may be processed and stored by the private home automation system for operational purposes.
  • Storage is limited to what is necessary for system functionality.
  • Data is retained only as long as required by the household system configuration.

Data sharing

  • Data is not shared with any third parties.
  • Data is not sold, rented, or transferred to external organisations.
  • Access is restricted to the household system operator(s).

User control and consent

  • Access to Google Nest device data is granted explicitly through Google’s OAuth consent flow.
  • Consent can be revoked at any time by removing the integration via the Google Home app or Google account security settings.

Security

Reasonable technical and organisational measures are taken to protect access credentials and any stored data from unauthorised access.

Children’s privacy

This system is not intended for use by children and does not knowingly collect personal information from children.

Changes to this policy

This privacy policy may be updated from time to time. Updates will be published on this page.
