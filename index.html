crowHome

crowHome is a private, non-commercial home automation system used within a single household.

What the app does

crowHome integrates with supported smart home platforms to monitor and control household devices. This includes integration with Google Nest devices via the Google Smart Device Management (SDM) API.

Use of data

The app requests access to Google Nest device data solely to enable home automation and monitoring features for the household that operates the system.

  • Data access is limited to supported device information exposed by the SDM API.
  • Data is used only for private home automation purposes.
  • No data is shared with third parties.
  • The app is not used for advertising or commercial purposes.
